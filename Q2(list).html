About Types of List in Html

1) Ordered list

In HTML, an ordered list is created using the <ol> tag, and each item is defined with <li>. It displays items in sequence,
and the type attribute can change numbering styles (1, A, i, etc.).

Example

  1. numbers (type="1")
  2. uppercase letters (type="A")
  3. lowercase letters (type="a")
  4. uppercase Roman numerals (type="I")
  5. lowercase Roman numerals (type="i")

2) Un-ordered list

An unordered list is created using the <ul> tag, with items defined by <li>. It shows items with bullets, which can be changed
using the type attribute.

Example

  • disc (type="disc")
  • circle (type="circle")
  • square (type="square")
  • none (type="none")
